# 한국항공대학교 공강 시간표

## Requirements
* `python>=3.4`
* `requests`
* `selenium`
* `flask`

## Usage
```
$ cd src/
$ python crawl1.py  # crawl data from university server

... wait while crawling is done ...

$ python crawl2.py  # manipulate data
$ python server.py 8080  # server is on. type 127.0.0.1:8080 on your webbrowser.
```
